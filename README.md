## My Projects  

### 🔐 [Microsoft Sentinel SOC Lab - Brute Force Detection](https://github.com/sohamsidral/sentinel-brute-force-detection-lab)  

_A practical SOC Analyst L1 lab focused on detecting multiple failed Windows login attempts using Microsoft Sentinel._

**Features:**  
- Created Azure Resource Group and Log Analytics Workspace  
- Enabled Microsoft Sentinel for SIEM monitoring  
- Deployed Windows Server endpoint  
- Collected Windows Security Events  
- Validated log ingestion using KQL  
- Created analytics rule for Event ID 4625  
- Generated and investigated Microsoft Sentinel incident  
- Mapped detection to MITRE ATT&CK Credential Access  

**Skills Demonstrated:**  
SIEM Monitoring, KQL, Log Analysis, Windows Event ID Analysis, Incident Investigation, Detection Rule Creation, MITRE ATT&CK Mapping
